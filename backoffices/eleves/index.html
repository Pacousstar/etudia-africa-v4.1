Questions Fréquentes

Comment fonctionne le système de parrainage ?

Le système de parrainage ÉtudIA vous permet de gagner des commissions sur 3 niveaux :

  • Niveau 1 (40%) : Commission directe sur vos filleuls
  • Niveau 2 (20%) : Commission sur les filleuls de vos filleuls
  • Niveau 3 (10%) : Commission sur le 3ème niveau

Partagez votre lien unique et gagnez à chaque nouvel abonnement !

Quels sont les différents modes IA disponibles ?

ÉtudIA V4.1 propose 4 modes d'apprentissage avec DeepSeek R1 :

  • 💬 Mode Normal (250 tokens) : Conversation équilibrée avec explications
  • 📚 Étape par Étape (180 tokens) : Résolution progressive et pédagogique
  • ⚡ Solution Directe (400 tokens) : Réponse complète et immédiate
  • 🎤 Mode Audio (200 tokens) : Réponses optimisées pour la synthèse vocale

Comment analyser mes documents PDF ?

Pour analyser vos documents :

  1. Allez dans la section "Mes Documents"
  2. Glissez-déposez votre PDF ou cliquez pour sélectionner
  3. Attendez le traitement OCR automatique
  4. Utilisez le Chat IA avec le contexte du document
  5. Posez vos questions sur le contenu !

Comment upgrader mon abonnement ?

Pour upgrader votre plan :

  1. Allez dans "Mon Abonnement"
  2. Consultez les plans disponibles
  3. Cliquez sur "Upgrader" pour le plan souhaité
  4. Effectuez le paiement via Mobile Money
  5. Profitez immédiatement des nouvelles fonctionnalités !
